# AIMAOS 작업 요약 보고서

작성일: 2026년 6월 7일

## 1. 프로젝트 목적

AIMAOS는 AI마케팅협회가 광고 운영 데이터를 수집, 표준화, 분석, 보고서화하고 장기적으로 지식 자산과 벤치마크 자산을 축적하기 위한 운영 시스템입니다.

이번 작업에서는 전체 플랫폼을 한 번에 만들지 않고, 가장 빨리 현장에서 쓸 수 있는 MVP 1인 “광고 데이터 분석기”를 먼저 구축했습니다.

## 2. 내가 기획한 MVP 방향

처음부터 광고 플랫폼 API나 대규모 SaaS를 만들지 않고, 광고센터에서 내려받은 엑셀 또는 CSV 파일을 분석하는 방식으로 설계했습니다.

이유는 다음과 같습니다.

- 광고 플랫폼 API는 제한적이거나 실무 활용성이 낮을 수 있음
- 현장에서는 엑셀 다운로드 데이터가 가장 빠르게 확보됨
- 분석 보고서 자동화가 가장 빠른 수익화 지점임
- 원본 파일, 분석 결과, 실행 권고가 쌓이면 협회 데이터 자산이 됨

## 3. 지금까지 만든 핵심 기능

### 3.1 광고 데이터 분석 파이프라인

입력 파일을 넣으면 아래 순서로 자동 처리됩니다.

1. 엑셀 또는 CSV 파일 읽기
2. 매체별 컬럼명을 공통 기준으로 자동 매핑
3. 숫자, 날짜, 텍스트 데이터 정리
4. 성과 지표 계산
5. 이상징후 탐지
6. 실행 권고 생성
7. Markdown, TXT, Excel 보고서 생성

주요 코드 위치:

```text
aimaos/pipeline.py
aimaos/parsers/excel_loader.py
aimaos/transformers/standardizer.py
aimaos/analyzers/performance_analyzer.py
aimaos/recommenders/rule_engine.py
aimaos/reports/report_generator.py
```

## 4. 분석 지표

현재 자동 계산되는 주요 지표는 다음과 같습니다.

| 한글 항목 | 의미 |
| --- | --- |
| 노출수 | 광고가 사용자 화면에 보인 횟수 |
| 클릭수 | 사용자가 광고를 클릭한 횟수 |
| 광고비 | 사용한 광고 비용 |
| 전환수 | 구매, 문의, 신청 등 목표 행동 횟수 |
| 매출 | 광고를 통해 발생한 매출 또는 전환 가치 |
| 주문수 | 광고를 통해 발생한 주문 수 |
| 클릭률 | 클릭수 ÷ 노출수 |
| 클릭당 비용 | 광고비 ÷ 클릭수 |
| 전환율 | 전환수 ÷ 클릭수 |
| 전환당 비용 | 광고비 ÷ 전환수 |
| 광고수익률 | 매출 ÷ 광고비 |

## 6. 초보자용 사용성 개선

초보 광고주나 회원사도 이해하기 쉽게 아래 작업을 적용했습니다.

- 엑셀 시트명을 모두 한글로 변경
- 엑셀 컬럼명을 모두 한글로 변경
- 보고서 지표명도 한글로 변경
- 업로드 화면 지표명도 한글로 변경
- 각 엑셀 헤더 셀에 설명 메모 추가
- 모든 시트에 필터 적용
- 모든 시트에 틀고정 적용
- 숫자는 1,000 단위 쉼표 적용
- 비율은 0.00% 형식 적용
- 광고수익률은 0.00배 형식 적용

예시:

```text
노출수: 1,603,400
광고비: 26,470,000원
클릭률: 1.41%
광고수익률: 4.15배
```

## 9. 현재 테스트 데이터에 들어간 의도

테스트 데이터에는 일부러 여러 상황을 섞었습니다.

| 캠페인 유형 | 의도 |
| --- | --- |
| 브랜드 검색 | 성과가 좋은 캠페인 예시 |
| 쇼핑 검색 | 예산 확대 후보 예시 |
| 경쟁사 키워드 | 광고비는 쓰지만 전환이 없는 위험 예시 |
| 디스플레이 신규유입 | 노출은 많지만 클릭률이 낮은 예시 |
| 디스플레이 리타겟팅 | 안정적인 전환 캠페인 예시 |
| 신규 확장 테스트 | 검토 또는 정리 후보 예시 |

```text
outputs/aimaos_mvp/.venv
```

설치된 주요 패키지:

- watchdog
- python-dotenv

확장 테스트는 실제로 실행했고, 보고서와 엑셀 파일 생성까지 확인했습니다.

## 12. 내가 실제로 해결한 문제

작업 중 발생한 문제와 처리 결과입니다.

| 문제 | 처리 |
| --- | --- |
| Python PATH 미설정 | 직접 Python 설치 경로를 찾아 가상환경 생성 |
| 패키지 설치 네트워크 제한 | 승인 후 필요한 패키지 설치 |
| 전환수 0일 때 나눗셈 오류 | 0 나눗셈 안전 처리 로직 추가 |
| 엑셀 파일 열림으로 덮어쓰기 실패 | 파일이 잠겨 있으면 시간표시가 붙은 새 엑셀 생성 |
| 결과가 어디 생성되는지 헷갈림 | 실행 파일이 최신 결과 파일을 자동으로 열게 수정 |
| 영어 컬럼명으로 초보자가 이해하기 어려움 | 모든 사용자 표시 영역을 한글화 |
| 지표 의미가 어려움 | 엑셀 헤더 셀에 설명 메모 추가 |
