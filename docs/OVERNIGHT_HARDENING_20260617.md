# CampaignPulse Overnight Hardening 2026-06-17

## 목적

Streamlit Cloud 온라인 데모에서 예시 데이터가 보이지 않는 문제와 라이트/다크 가독성 문제를 해결하고, 실제 광고주 데이터 없이도 CampaignPulse의 전체 흐름을 확인할 수 있게 했다.

## 실제 반영

- 안전한 가짜 데모 데이터 3개 광고주, 4개 매체
- 데모 보고서 Excel / Markdown / TXT
- 쇼핑몰/검색/AI 가시성 예시 데이터
- Streamlit 라이트 테마 기본값과 다크 환경 대비 보정
- 데모 데이터 기본 로딩과 사이드바 토글
- 화면 상단 데모 데이터 안내
- 실제 파일 분석 시 업로드 데이터 우선 적용
- GitHub Actions 배포 준비 검사
