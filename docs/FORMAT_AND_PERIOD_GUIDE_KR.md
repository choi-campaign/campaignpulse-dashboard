# 표시 형식과 기간별 분석 기준

## 표시 형식

AIMAOS는 계산값은 숫자로 유지하고, 보고서와 화면에서만 보기 좋은 형식으로 표시합니다.

| 구분 | 예시 | 적용 위치 |
| --- | --- | --- |
| 일반 숫자 | `1,603,400` | 노출수, 클릭수, 전환수, 주문수 |
| 금액 | `26,470,000원` | Markdown 보고서, 업로드 화면 |
| 엑셀 금액 | `26,470,000` | 엑셀 셀, 계산 가능 숫자 유지 |
| 비율 | `1.41%` | 클릭률, 전환율, 증감률 |
| 광고수익률 | `415.00%` | 광고수익률 |

## 기간별 분석 시트

엑셀 결과 파일에는 아래 시트가 자동 생성됩니다.

| 시트명 | 의미 |
| --- | --- |
| `일자별성과` | 일자별 성과 |
| `주간별성과` | 주간별 성과 |
| `월별성과` | 월별 성과 |
| `분기별성과` | 분기별 성과 |
| `반기별성과` | 반기별 성과 |
| `연도별성과` | 연도별 성과 |
| `시즌별성과` | 시즌별 성과 |

## 기간별 공통 컬럼

각 기간별 시트는 아래 컬럼을 포함합니다.

- 기간
- 시작일
- 종료일
- 행 수
- 노출수
- 클릭수
- 광고비
- 전환수
- 매출
- 주문수
- 클릭률
- 클릭당 비용
- 전환율
- 전환당 비용
- 광고수익률

## 시즌 기준

시즌은 월 기준으로 나눕니다.

- 봄: 3월, 4월, 5월
- 여름: 6월, 7월, 8월
- 가을: 9월, 10월, 11월
- 겨울: 12월, 1월, 2월

## 사용 팁

먼저 `월별성과`와 `주간별성과`를 보고 전체 흐름을 확인한 다음, 문제가 있는 기간을 `일자별성과`에서 자세히 보는 방식이 가장 빠릅니다.
