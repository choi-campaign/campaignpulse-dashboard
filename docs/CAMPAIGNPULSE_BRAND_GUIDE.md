# 캠페인펄스 CampaignPulse 브랜드 가이드

작성일: 2026-06-13

## 1. 브랜드명

- 한글명: 캠페인펄스
- 영문명: CampaignPulse
- 권장 표기: 캠페인펄스 CampaignPulse

## 2. 브랜드 정의

캠페인펄스는 광고 캠페인의 성과 신호를 읽고, 운영자가 지금 확인해야 할 일을 정리해 주는 광고 운영 대시보드입니다.

단순히 광고 데이터를 보여주는 보고서 생성기가 아니라, 광고주와 운영자가 매일 확인할 수 있는 운영 판단 화면을 목표로 합니다.

## 3. 핵심 문장

광고 성과의 맥박을 읽고, 지금 해야 할 일을 보여줍니다.

## 4. 로고 방향

로고 컨셉은 `Pulse Signal Dashboard`입니다.

- 둥근 모서리의 대시보드 카드
- 광고 성과 변화를 나타내는 부드러운 신호선
- 주요 변화 지점을 나타내는 Pulse Cyan 포인트
- 하단 막대 그래프 3~4개
- 작은 크기에서도 알아볼 수 있는 단순 구조

참고 PNG는 브랜드 방향 확인용이며, 실제 화면에는 새로 만든 SVG 로고를 사용합니다.

## 5. 로고 파일

현재 생성된 SVG 파일:

- `public/assets/brand/campaignpulse-logo.svg`
- `public/assets/brand/campaignpulse-logo-dark.svg`
- `public/assets/brand/campaignpulse-icon.svg`
- `public/assets/brand/campaignpulse-favicon.svg`

사용 기준:

- 밝은 배경: `campaignpulse-logo.svg`
- 어두운 사이드바: `campaignpulse-logo-dark.svg`
- 작은 로고 영역: `campaignpulse-icon.svg`
- 브라우저 탭/즐겨찾기 후보: `campaignpulse-favicon.svg`

## 7. 폰트 기준

한글 UI 우선순위:

1. Pretendard
2. Noto Sans KR
3. SUIT
4. system-ui
5. sans-serif

영문/숫자 우선순위:

1. Inter
2. Roboto
3. system-ui
4. sans-serif

숫자 지표는 크게, 명확하게 표시하고 설명 문구는 짧고 보조적으로 사용합니다.

## 8. UI 문구 원칙

- 데이터를 보여주는 표현보다 다음 행동을 안내하는 표현을 우선합니다.
- 사용자를 가르치는 말투보다 운영자가 바로 판단할 수 있는 말투를 사용합니다.
- 원인 표현은 단정하지 않고 가능성, 의심, 검토 필요 수준으로 씁니다.
- 데이터가 부족하면 부족하다고 말합니다.
- 개발자 용어는 화면에서 줄입니다.

권장 표현:

- 데이터 상태 센터
- 오늘 해야 할 일
- 우선 처리 이슈
- 오늘의 운영 브리핑
- 광고 데이터 업로드
- 보고서 만들기
- 데이터 부족
- 데이터 확인 필요
- 계정 연동 수집

피해야 할 표현:

- MVP
- P0
- Rule Engine
- selector
- profile
- Playwright
- 자동 다운로드 에이전트
- 내부 설계 용어

## 9. 상태 표현 기준

| 상태 | 색상 | 의미 | 예시 문구 |
| --- | --- | --- | --- |
| 개선 | `#16A34A` | 성과 개선, ROAS 상승, 전환율 개선 | ROAS가 이전 기간 대비 개선되었습니다. |
| 주의 | `#F59E0B` | 변동성 증가, 확인 필요 | 성과 변동 폭이 커졌습니다. 최근 변경사항을 확인하세요. |
| 위험 | `#DC2626` | 광고비 급증, 전환 급락, 성과 악화 | 광고비가 급증했습니다. 입찰가 또는 예산 변경 여부를 확인하세요. |
| 데이터 부족 | `#94A3B8` | 판단 불가, 기간 부족, 데이터 누락 | 현재 데이터만으로는 판단하기 어렵습니다. |
| 확대기회 | `#06B6D4` | 예산 확대 후보, 성장 가능 구간 | 성과 개선 신호가 확인되었습니다. 예산 확대 가능성을 검토하세요. |
