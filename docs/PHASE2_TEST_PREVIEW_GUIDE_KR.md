# Phase 2 테스트 미리보기 안내

## 1. 목적

이 문서는 AIMAOS Phase 2 확장 메뉴를 테스트로 확인하는 방법을 설명합니다.

현재 단계에서는 실제 크롤러, 로그인 자동화, 데이터 수집을 구현하지 않았습니다. 화면에 보이는 데이터는 구조와 사용성을 확인하기 위한 샘플 데이터입니다.

## 2. 테스트 가능한 메뉴

Streamlit 앱에서 아래 메뉴를 확인할 수 있습니다.

```text
Commerce Intelligence
Search Intelligence
AI Visibility
```

각 메뉴에는 다음 내용이 표시됩니다.

- 모듈 목적
- 대상 플랫폼
- 향후 저장 예정 데이터
- 향후 설계 방향
- 테스트 샘플 미리보기
- 현재 단계에서 하지 않는 일

## 3. 샘플 데이터 위치

테스트 미리보기용 샘플 데이터는 아래 폴더에 있습니다.

```text
data/phase2_samples
```

파일 구성:

| 파일 | 설명 |
| --- | --- |
| `commerce_sample.csv` | 커머스 상품, 가격, 리뷰, 평점, 경쟁상품 예시 |
| `search_sample.csv` | 검색 키워드, 검색량, 경쟁도, 경쟁사 노출 예시 |
| `ai_visibility_sample.csv` | AI 답변 내 브랜드 언급, 경쟁사 언급, 추천 여부 예시 |

## 4. Commerce Intelligence 테스트 화면

확인할 수 있는 예시 항목:

- 스마트스토어
- 쿠팡
- G마켓
- 옥션
- 11번가
- 상품명
- 가격
- 리뷰수
- 평점
- 판매량
- 경쟁상품

## 5. Search Intelligence 테스트 화면

확인할 수 있는 예시 항목:

- 네이버
- 구글
- 카카오다음
- 네이트
- 키워드
- 검색량
- 경쟁도
- 검색 결과 유형
- 경쟁사 노출

## 6. AI Visibility 테스트 화면

확인할 수 있는 예시 항목:

- Claude
- Gemini
- Perplexity
- 질문
- 브랜드 언급
- 경쟁사 언급
- 추천 여부
- 답변 출처

## 7. 현재 단계에서 하지 않는 일

아래 작업은 일부러 구현하지 않았습니다.

- 실제 크롤러 구현
- 실제 로그인 자동화 구현
- 실제 데이터 수집 실행
- 광고 분석 엔진 변경
- 캡차 우회
- 보안 우회
- 정책 위반 가능성이 있는 자동화

## 8. 테스트 방법

1. `start_upload_app.bat`을 실행합니다.
2. 브라우저에서 AIMAOS 화면이 열리면 왼쪽 메뉴를 확인합니다.
3. `Commerce Intelligence`, `Search Intelligence`, `AI Visibility` 메뉴를 각각 클릭합니다.
4. `테스트 샘플 미리보기` 표가 보이면 정상입니다.
5. 기존 광고 데이터 분석 기능은 `광고 데이터 분석기` 메뉴에서 그대로 사용할 수 있습니다.

## 9. 정상 동작 기준

정상이라면 아래가 보여야 합니다.

- Commerce Intelligence 화면에서 스마트스토어, 쿠팡, G마켓 등이 보임
- Search Intelligence 화면에서 네이버, 구글, 카카오다음 등이 보임
- AI Visibility 화면에서 ChatGPT, Claude, Gemini 등이 보임
- 각 화면에 “실제 크롤러 구현”을 하지 않는다는 안내가 보임
- 광고 데이터 분석기 메뉴에서 기존 엑셀/CSV 업로드 화면이 그대로 보임
